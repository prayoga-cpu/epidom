# Epidom Growth Strategy (Based on SaaS Best Practices)

Strategi pertumbuhan jangka panjang Epidom ini disusun berdasarkan prinsip-prinsip pemasaran SaaS yang terstruktur (mengacu pada panduan Hostinger), fokus pada skalabilitas, metrik terukur, dan Product-Led Growth.

## 1. Product-Led Growth (Strategi Produk sebagai Marketing)

Biarkan produk menjual dirinya sendiri dengan menurunkan hambatan masuk (_barrier to entry_).

- **Freemium Model Optimization:**
  - Pastikan paket **STARTER** Epidom benar-benar gratis (Free Forever) dengan batasan kuota (misal: max 50 resep), bukan batasan waktu (trial 14 hari).
  - _Tujuan:_ Akuisisi user sebanyak-banyaknya di awal (Top of Funnel).
- **Interactive "Playground" Demo:**
  - Buat halaman demo interaktif di website tanpa perlu login/signup.
  - Biarkan pengunjung mencoba simulasi: Input harga telur -> Lihat HPP berubah otomatis dalam 10 detik.
  - _Tujuan:_ Membunuh keraguan user ("Apakah aplikasi ini susah?") sebelum mereka mendaftar.

## 2. Onboarding & Activation (Mengejar "Aha Moment")

Sukses marketing bukan saat user _Sign Up_, tapi saat user merasakan manfaat pertama (_Activation_).

- **Dummy Data (Toko Contoh):**
  - Saat user baru login, dashboard tidak boleh kosong melompong. Sediakan opsi _"Load Sample Data"_ (Toko Roti Contoh) agar user langsung paham visual laporan dan grafik.
- **In-App Checklist:**
  - Tampilkan progress bar di dashboard user baru:
    1.  [ ] Buat Store
    2.  [ ] Input 1 Bahan Baku
    3.  [ ] Buat 1 Resep (Goal Utama/Aha Moment)
- **Email Automation:**
  - **Hari 1:** "Selamat datang! Ini video 1 menit cara hitung HPP pertamamu."
  - **Hari 3:** "Stok tepung menipis? Ini cara aktifkan fitur Low Stock Alert."

## 3. SEO & Content Marketing (Long-Term Investment)

Membangun aset digital untuk menangkap pencarian organik (Inbound Marketing).

- **High Intent Keywords (Artikel Blog):**
  - Tulis artikel yang menjawab masalah teknis spesifik, bukan topik umum.
  - Contoh: _"Cara menghitung HPP kue kering agar tidak rugi"_, _"Rumus Food Cost untuk Pabrik Roti"_, _"Aplikasi inventory vs Excel untuk stok gudang"_.
- **Engineering as Marketing (Free Tools):**
  - Buat tools web sederhana gratisan, misal: **"Kalkulator Food Cost Sederhana"**.
  - User pakai tool gratis -> Suka -> Ditawari Epidom untuk fitur yang lebih lengkap (misal: history harga, multi-store).

## 4. Ads & Retargeting (Paid Channel)

Menangkap demand yang sudah ada (High Intent) untuk hasil cepat.

- **Google Ads (Search):**
  - Target kata kunci spesifik: _"Software ERP Pabrik Makanan"_, _"Aplikasi Kasir Toko Roti dengan Inventory"_.
  - Hindari kata kunci umum _"Aplikasi Bisnis"_ (Terlalu mahal, konversi rendah).
- **Retargeting:**
  - Tampilkan iklan visual di sosmed kepada orang yang sudah mengunjungi halaman harga (Pricing Page) tapi belum daftar. Tawarkan insentif diskon atau demo gratis.

## 5. Partnership & Ecosystem (Affiliate)

Leverage kolam audiens partner.

- **Supplier Partnership:**
  - Kerjasama dengan Supplier Bahan Kue/Tepung/Packaging.
  - Tawaran: _"Beli 10 sak tepung X, gratis langganan Epidom Pro 1 bulan."_ (Win-win: Supplier dapat nilai tambah, Epidom dapat user tertarget).
- **Consultant Affiliate:**
  - Ajak Konsultan Bisnis Kuliner / Chef Consultant. Berikan komisi 20-30% jika mereka merekomendasikan Epidom ke klien restoran/bakery mereka.

## 6. Key Metrics Monitoring (Kesehatan Bisnis)

Marketing tanpa data adalah buta. Pantau metrik ini secara rutin.

- **CAC vs LTV:** Pastikan biaya akuisisi (_Cost per Acquisition_) lebih rendah dari nilai jangka panjang (_Lifetime Value_) user.
- **Activation Rate:** % User yang berhasil membuat minimal 1 Resep dalam minggu pertama.
- **Churn Rate:** % User berbayar yang berhenti langganan. Jika tinggi, prioritaskan perbaikan produk/bug fixing daripada menambah iklan.
