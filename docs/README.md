# EPIDOM - Technical Documentation

## Overview

EPIDOM is an open-source ERP (Enterprise Resource Planning) system designed specifically for small food manufacturers, bakeries, and cookie bars. It provides comprehensive inventory management, production tracking, and supplier management capabilities.
